# About
```
This tool is made to scan websites that are vulnerable to sqli and scan admin logins.
```
# Requirements :

```
- python 2.7
- requests
- bs4
```
# Installation :

```
$ pkg install python2
$ pkg install git
$ pip2 install requests
$ pip2 install bs4
$ git clone https://github.com/Fukur0-3XP/Sqli
$ cd Sqli
$ python2 botscan.py
```
